# Tourney #
A round-robin tournament scheduler.  This is especially useful for creating
tennis tournaments with doubles teams.

### Build Requirements ###
* `rust` (from git)
* `ninja`
* `qt4` or the Android SDK

### Build Instructions ###
```bash
$ ./configure
$ ninja
```
